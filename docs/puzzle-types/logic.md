# Logic Puzzles

## Grid Deduction (Nonograms, Picross) { #nonogram }

**Identify by:** A grid with number clues on rows and columns indicating runs of filled cells.

### Tools
- [Nonogram solver](https://www.nonosolver.com/)
- [Picross solver](https://www.griddlers.net/nonogram-solver)

---

## Sudoku Variants { #sudoku }

Common variants in puzzlehunts:
- **Killer Sudoku** — cages with sums instead of givens
- **Thermometer Sudoku** — cells must increase along thermometer shapes
- **Arrow Sudoku** — digits on arrows sum to the circle
- **Knights-move Sudoku** — no same digit a knight's move away

### Tools
- [f-puzzles](https://f-puzzles.com) — build and solve custom sudoku variants
- [Sudoku solver](https://sudoku.com/solver/)

## Logic Grid Puzzles { #logic-grid }

## Constraint Satisfaction

Many hunt puzzles are constraint satisfaction problems in disguise. If you have:
- A set of items
- A set of constraints (X must go before Y, A and B can't be together, etc.)

Try writing out a simple elimination grid.

---

## Common Puzzle Hunt Logic Structures

- **Masyu** — circles on a grid, white/black have different path rules
- **Slitherlink** — draw a loop using clue numbers
- **Hashi (Bridges)** — connect islands with bridges
- **Nurikabe** — flood-fill logic
- **Fillomino** — partition grid into polyominoes

All solvable at [Puzzle Picnic](https://www.puzzlepicnic.com/) or [Simon Tatham's Puzzles](https://www.chiark.greenend.org.uk/~sgtatham/puzzles/).
