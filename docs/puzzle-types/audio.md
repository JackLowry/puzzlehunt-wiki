# Audio Puzzles

---

## Spectrograms { #spectrogram }

**Identify by:** Audio that sounds like noise or music but may have a visual image or text hidden in the frequency domain.

### How to analyze
Open in Sonic Visualiser, add a Spectrogram layer:
- Set window size: 2048 or 4096
- Set color scheme: high contrast (White on Black)
- Zoom in on the frequency axis

### Tools
- [Sonic Visualiser](https://www.sonicvisualiser.org) — best tool for this
- Audacity → View → Spectrogram view

---

## Backwards Audio { #backwards }

**Identify by:** Audio that sounds garbled or reversed. Try reversing it.

### Tools
- Audacity: Effect → Reverse

---

## Speed / Pitch Changes { #speed }

**Identify by:** Audio that sounds pitch-shifted or sped up/slowed down.

- Try 50% speed (slow down) to hear hidden speech
- Try 2× speed to compress long recordings
- Pitch shift separately from speed (Audacity → Effect → Change Pitch)

---

## Morse in Audio { #morse-audio }

**Identify by:** Beeps at two lengths (short/long) with pauses.

- Short = dot, long = dash
- Gap between letters = longer silence
- Transcribe to dots/dashes, then decode with [Morse](../symbol-languages/morse.md)

---

## DTMF (Phone Tones) { #dtmf }

**Identify by:** Sounds like dialing a phone — each tone is a unique frequency pair.

### Tools
- [DTMF Decoder](https://www.dialabc.com/sound/detect/) — upload audio
- [Online DTMF Decoder](http://www.dtmf.de/)

## Music & Notes { #music }

**Identify by:** Sheet music, a melody, or note names used as letters.

Notes A–G can spell short words. See [Musical Notes](../indexing/other-orderings.md#music) for details.
