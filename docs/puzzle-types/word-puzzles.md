# Word Puzzles

---

## Anagrams { #anagram }

**Identify by:** Letters that can be rearranged. Often clued by "mixed up", "scrambled", or an unusual arrangement of letters.

### Tools
- [dcode.fr Anagram Solver](https://www.dcode.fr/anagram-solver)
- [Internet Anagram Server](https://wordsmith.org/anagram/) — good for multi-word anagrams
- [Nutrimatic](https://nutrimatic.org) — use `*` patterns

---

## Homophones { #homophone }

**Identify by:** Words that sound like other words. Clued by "sounds like", "say", "spoken", "heard".

A=Ay, B=Be/Bee, C=Sea/See, I=Eye, K=Kay, O=Oh/Owe, P=Pee, Q=Queue/Cue, R=Are, T=Tea, U=You, Y=Why

---

## Acrostics { #acrostic }

**Identify by:** First (or last, or nth) letters of words or lines spell something.

Common extraction: take the first letter of each answer in a grid, read in order.

---

## Spoonerisms { #spoonerism }

**Identify by:** Swapping the initial consonant sounds of two words.
`WEAK PEAS` → `PEAK WEAS` (pea wax? — think "Weak peas" → "Peak wise"...)

---

## Wordplay with Numbers

- **Split words:** `THERAPIST` = `THE RAPIST` — finding hidden words with a space shift
- **Roman numerals in words:** `CIVIL` contains `CIV` = 104, `IVY` contains `IV` = 4
- **Words containing element symbols:** `BEACH` contains `BE` (Beryllium, #4)

---

## Rebuses { #rebus }

**Identify by:** Pictures + letters/sounds combined to make words.
Eye + Land = Island. Bee + 4 = Before.

---

## Tools

- [OneLook](https://www.onelook.com) — dictionary + reverse dictionary + wildcard
- [Nutrimatic](https://nutrimatic.org) — `A*B` = words starting with A, ending with B
- [Qat](https://www.quinapalus.com/qat.html)
