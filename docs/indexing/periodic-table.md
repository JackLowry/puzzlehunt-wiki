# Periodic Table

**Tags:** `elements` `atomic number` `symbol`  
**Identify by:** Element names, symbols (1–2 letters like Fe, Au, Hg), or atomic numbers (1–118) appearing in a puzzle context.

## Two common encodings

| Encoding | Example |
|----------|---------|
| Symbol → letter(s) | `He` = H+E, `Fe` = F+E (spells letters) |
| Atomic number → A1Z26 | Element #8 (Oxygen) → H |
| Element name → index | First letter of element name |

## Useful elements for puzzles

Elements whose symbols differ from their names are especially common:

| Element | Symbol | Atomic # |
|---------|--------|----------|
| Iron | Fe | 26 |
| Gold | Au | 79 |
