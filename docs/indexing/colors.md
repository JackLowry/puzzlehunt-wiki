# Colors & Visual Orderings

---

## ROYGBIV (Rainbow / Visible Spectrum) { #roygbiv }

**Tags:** `rainbow` `spectrum` `colors`

```
R = Red        (1)
O = Orange     (2)
Y = Yellow     (3)
G = Green      (4)
B = Blue       (5)
I = Indigo     (6)
V = Violet     (7)
```

Used for ordering or indexing when colors appear in a puzzle.

## HTML / Hex Colors { #hex-color }

**Tags:** `#RRGGBB` `hex color` `web colors`  
**Identify by:** 6-digit hex strings that might be color codes. The color itself or its name might be the clue.

Named colors with useful letter properties:
- `RED`, `TAN`, `Teal`, `BLUE`, `LIME`, `GOLD`, `ROSE`, `CYAN`, `NAVY`...
