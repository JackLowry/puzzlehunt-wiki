# Symbol Ciphers

Ciphers that replace letters with drawn symbols or geometric shapes.

---

## Pigpen Cipher { #pigpen }

**Tags:** `grid` `geometric` `masonic`  
**Identify by:** Symbols made of lines forming parts of a tic-tac-toe grid or X, sometimes with a dot inside.

### How to decode
Two grids: a `#` shape and an `X` shape, each split into 4 sections. First pass = no dot, second pass = dot.

```
#-grid (no dot):  A B C / D E F / G H I
#-grid (dot):     J K L / M N O / P Q R
X-grid (no dot):  S T / U V
X-grid (dot):     W X / Y Z
```

Each symbol is the outline of its grid section.

### Tools
- [dcode.fr Pigpen](https://www.dcode.fr/pigpen-cipher)

## Templar Cipher { #templar }

**Tags:** `grid` `cross` `variant`  
**Identify by:** Similar to Pigpen but uses a cross/plus arrangement. Sometimes called "Tic-Tac-Toe cipher" variant.

### Tools

## Wingdings / Webdings { #wingdings }

**Tags:** `font` `symbols` `MS-Office`  
**Identify by:** Symbols that look like clip-art or icons — could be a font substitution cipher.

### How to decode
Paste into a text editor and change the font to Wingdings, Wingdings 2, Wingdings 3, or Webdings to see what letters they map to.

### Tools
- [Wingdings Translator](https://www.wingdings.net/)
- [dcode.fr Wingdings](https://www.dcode.fr/wingdings-font)
