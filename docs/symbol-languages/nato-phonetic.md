**Tags:** `military` `aviation` `spelling alphabet` `words`

## Common variants

Some hunts use older or alternate phonetic alphabets (ABLE BAKER CHARLIE…). If the standard NATO words don't fit, check the [ICAO/ITU history](https://en.wikipedia.org/wiki/NATO_phonetic_alphabet#History).
