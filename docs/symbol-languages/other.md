## Tap Code { #tap }

**Tags:** `knock` `prison` `5x5 grid`  
**Identify by:** Sequences of taps or knocks in pairs. C and K are merged (C is used for both).

### Grid
```
     1    2    3    4    5
1    A    B   C/K   D    E
2    F    G    H    I    J
3    L    M    N    O    P
4    Q    R    S    T    U
5    V    W    X    Y    Z
```
Each letter = (row taps) pause (column taps). E.g. H = 2 taps, pause, 3 taps.

## Maritime Signal Flags { #maritime }

**Tags:** `nautical` `flags` `colored`  
**Identify by:** Colorful rectangular or triangular flags on ships; each flag = one letter or digit.

### Tools
- [dcode.fr Maritime Flags](https://www.dcode.fr/maritime-signal-flags)

## Resistor Color Codes { #resistor }

**Tags:** `electronics` `colors` `bands`  
**Identify by:** Color sequences clued by electronics context. Colors encode digits 0–9.

```
Black=0  Brown=1  Red=2  Orange=3  Yellow=4
Green=5  Blue=6   Violet=7  Gray=8  White=9
```

### Tools
- [Resistor Color Code Calculator](https://www.digikey.com/en/resources/conversion-calculators/conversion-calculator-resistor-color-code)

## Polybius Square { #polybius }

**Tags:** `5x5` `grid` `coordinate pairs`  
**Identify by:** Pairs of numbers in range 1–5, often I and J are merged.

```
  1 2 3 4 5
1 A B C D E
2 F G H IJ K
3 L M N O P
4 Q R S T U
5 V W X Y Z
```

### Tools
- [dcode.fr Polybius](https://www.dcode.fr/polybius-square)
