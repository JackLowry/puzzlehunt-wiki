# Braille

**Tags:** `dots` `6-dot` `raised` `tactile`  
**Identify by:** 6-dot cell grid (2 columns × 3 rows), dots numbered 1–6. Often rendered as filled/empty circles or a 2×3 grid of dots.

## Grade 1 Braille Alphabet

Dot positions:
```
1 4
2 5
3 6
```

| Letter | Dots | Letter | Dots | Letter | Dots |
|--------|------|--------|------|--------|------|

## Numbers

Numbers use the same cells as A–J but preceded by a number indicator (`3,4,5,6`):
```
#1=A  #2=B  #3=C  #4=D  #5=E  #6=F  #7=G  #8=H  #9=I  #0=J
```
